ARMEEBOX V1 STARTER - Setup

1. Entpacken.
2. Datei .env.example kopieren als .env
3. In .env einfuegen:
   VITE_SUPABASE_URL=DEINE_URL
   VITE_SUPABASE_ANON_KEY=DEIN_SB_PUBLISHABLE_KEY
4. Projekt nach GitHub pushen.
5. In Netlify neues Projekt mit GitHub Repo verbinden.
6. Build command: npm run build
7. Publish directory: dist
8. Nach Deploy Domain spaeter verbinden.

Hinweis:
- Admin Login mit admin@armeebox.ch und dem Passwort aus Supabase Auth.
- Dies ist der erste deploybare Starter-Stand.
- E-Mail Versand und echte Zahlungsanbindung sind noch nicht live angeschlossen.
